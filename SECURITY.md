# Security Policy

## Responsible Disclosure

If you identify a security issue in this project, please report it privately to the maintainers before public disclosure.

Include:
- clear reproduction steps
- expected vs actual behavior
- impact assessment

## Scope and Safety

- This repository is for defensive security monitoring and detection research.
- Do not use this project to develop, test, or distribute malware.
- Do not submit malicious binaries, live payloads, or weaponized artifacts via issues/PRs.
