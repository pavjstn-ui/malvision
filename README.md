# MalVision

MalVision is a defensive behavioral malware/ransomware monitoring prototype. It combines host-side telemetry collectors with a FastAPI scoring engine and an operator dashboard.

Intended use: internal security detection research, triage, and alerting workflows.

## Status

Active prototype for lab and controlled environment usage.

## Features

- Filesystem monitor for entropy spikes, honeytoken access, ransom note patterns, and rename bursts
- Process monitor for suspicious command patterns, backup-kill behavior, and rapid file-open patterns
- Network monitor for SMB spread, connection burst anomalies, and suspicious destination ports
- FastAPI detection engine with host scoring, alerts, replay, and health/status endpoints
- Optional alert fan-out via webhook, email SMTP, and Splunk HEC
- Browser dashboard for near-real-time threat monitoring

## Supported Inputs

- Runtime host telemetry generated by:
  - `agent/watcher.py` (filesystem events)
  - `agent/process_monitor.py` (process activity)
  - `agent/net_monitor.py` (network connections)
- JSON event payloads sent to `POST /events`
- Local queued events in `event_queue.jsonl` for replay via `POST /replay`

## Quickstart

### Prerequisites

- Python 3.10+
- `pip`

### Install

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Run Engine

```bash
uvicorn engine.main:app --host 0.0.0.0 --port 8000 --reload
```

### Run Collectors (separate terminals)

```bash
python agent/watcher.py
python agent/process_monitor.py
python agent/net_monitor.py
```

### Basic API checks

```bash
curl http://127.0.0.1:8000/health
curl http://127.0.0.1:8000/status
curl http://127.0.0.1:8000/alerts
```

## Safety Notes

- Do not run suspicious binaries or live malware samples on your host OS.
- Use isolated lab infrastructure (VM/sandbox, segmented network, disposable snapshots).
- Test with synthetic/safe simulation data unless you have a formal malware handling process.
- This project is detection-oriented; it does not distribute malware or exploit tooling.

## Project Layout

- `agent/`: host telemetry collectors
- `engine/`: FastAPI scoring and alert pipeline
- `dashboard/`: operator UI
- `deploy/`: deployment helper scripts/config
- `Dockerfile`, `docker-compose.yml`: containerized run options
